# Wordle
Explicacion del juego: 
La esencia del juego es adivinar una palabra oculta de 5 letras en un máximo de 6 intentos. Cada vez que lanzas una palabra, el juego te devuelve información basada en  [], (), ''

[ ] La letra es correcta y esta en el lugar adecuado

( ) La letra esta en la palabra, pero en el lugar equivocado
 
' ' La letra no esta en la plabra
